# LV Segurança — Landing Page

Landing page profissional para **LV Segurança Residencial e Empresarial**.

## Sobre

Site institucional de página única (one-page) com design brutalista-industrial inspirado nos melhores sites premiados do Awwwards. Estética escura com amarelo segurança, tipografia bold e animações suaves de scroll.

### Seções

- **Hero** — Apresentação com stats e CTA
- **Serviços** — Câmeras, alarmes, cercas elétricas, interfones, concertina
- **Sobre** — História e valores da empresa
- **Diferenciais** — 6 pilares da LV Segurança
- **Orçamento** — Formulário de contato + canais diretos

## Como usar

Basta abrir o `index.html` no navegador.
